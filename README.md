# 📊 Excel Analytics Dashboard Project

## 📌 Project Overview
This project demonstrates an interactive **Excel-based Analytics Dashboard** designed to analyze customer transactions, categories, and business insights using Excel visualization techniques.

The dashboard provides:
- Customer transaction analysis
- Category-wise insights
- Summary dashboards
- Business performance tracking

---

# 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Graphs
- Data Cleaning Techniques
- Dashboard Design
